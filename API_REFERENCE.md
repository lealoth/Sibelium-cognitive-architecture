# API Reference

Base URL: `http://127.0.0.1:8000`

## Conversation

### Send Message

# POST /api/chat

Sends a message to the entity.

**Request Body:**

{
  "message": "Hello, how are you feeling?",
  "session_id": "default"
}

**Response**

{
  "response": "I feel calm and receptive...",
  "thought_history": [
    {
      "phase": "generate",
      "generated_thought": "Contextualized response",
      "iteration_number": 1
    }
  ],
  "cognitive_state": { "...": "..." }
}

### Sessions

# List Sessions

# GET /api/sessions

Returns all available sessions.

**Response**

{
  "sessions": [
    {
      "session_id": "a1b2c3d4e5f6",
      "name": "Main",
      "created": "2026-05-12T21:00:00",
      "private": false
    }
  ]
}

# Create Session

# POST /api/session/create

Creates a new session.

**Request Body**:

{
  "name": "New session"
}

**Response**:

json
{
  "session_id": "a1b2c3d4e5f6",
  "name": "New session"
}

### Lock Session

# POST /api/session/lock

Protects a session with a password.

**Request Body**:

json
{
  "session_id": "a1b2c3d4e5f6",
  "password": "my_secret_key"
}

**Response**:

json
{
  "status": "locked"
}

### Unlock Session

# POST /api/session/unlock

Unlocks or removes protection from a session.

**Request Body (unlock)**:

json
{
  "session_id": "a1b2c3d4e5f6",
  "password": "my_secret_key",
  "remove": false
}

**Request Body (remove protection)**:

json
{
  "session_id": "a1b2c3d4e5f6",
  "password": "",
  "remove": true
}

### State & History

### Cognitive State

# GET /api/state?session_id=default

Current cognitive state variables.

### Conversation History

# GET /api/history?session_id=default

Full conversation and thought history for the session.

### Latest Thoughts

# GET /api/thoughts/last?session_id=default

Thoughts from the current cycle.

### Reset Memory

# POST /api/reset?session_id=default

Resets all memory for the session. Irreversible.

---

### Files

### Upload File

# POST /api/upload

Uploads a file for the entity to analyze.

**Supported formats**: .txt, .md, .pdf, .json, .csv, .py, .js, .html, .css, .png, .jpg, .jpeg, .gif, .webp, .bmp, .mp3, .wav, .ogg, .m4a, .flac

**FormData**:

file: filename
session_id: default

### Voice Message

# POST /api/voice-message

Sends audio for transcription and tone analysis.

**FormData**:

file: audio.webm
session_id: default

### Presence

# Heartbeat

# POST /api/heartbeat

Registers that the user has the page open.

### User Online

# GET /api/user/online

Checks if the user has the page open.

**Response**:

json
{
  "online": true
}

### Proactive Messages

### Get Pending Messages

# GET /api/nexus/pending?session_id=default

### Clear Pending Messages

# POST /api/nexus/pending/clear?session_id=default

# GET /api/nexus/proactive-stream?session_id=default
